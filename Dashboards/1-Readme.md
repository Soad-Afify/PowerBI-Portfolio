# Sales Report:

### Data source:
- based on PowerBI course on Udemy

### Data model type:
- Star schema

### Report visualizartion:
- Measures are assigned to cards
- Horizontal filter by color
- Sales amount per year
- Sold products per country
- Sold products per gender
